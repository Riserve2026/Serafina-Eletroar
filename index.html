1. Acesse console.cloud.google.com
2. Crie um projeto → Ative Google Drive API e Google Picker API
3. Em "Credenciais" → crie uma Chave de API e um ID do cliente OAuth 2.0 (tipo: Aplicativo da web)
4. Na origem JavaScript autorizada, adicione http://localhost ou a URL onde usa o arquivo
